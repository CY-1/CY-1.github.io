一些知识点

Post View:

  • 语句sorted([1, 2, 3], reverse=True) == reversed([1, 2, 3])执行结果为False 前者返回一个列表 后者返回一个迭代器

  • n&1 n%2 都是判断是否为偶数 一个位运算 一个求余数 是偶数都返回0

  • locals() 查看当前所有局部和全局变量

  • 1<2<3 相当于 2>1 amd 2<3

  • and中含0，返回0； 均为非0时，返回后一个值 or中， 至少有一个非0时，返回第一个非0

    • 3 or 5 返回 3
    • 0 or 5 返回 5
    • 3 and 5 返回 5
    • 3 and not 5 返回 False

  • 可迭代对象下标取值超出索引范围,会报错:IndexError ,可迭代切片取值超出索引范围,不报错,而是返回对应的空值.

    • x=[1,3,4] x[4]报错 但是x[4:] 会返回空值

  • 表达式 list(zip([1,2], [3,4])) 的值为 [(1, 3), (2, 4)]

  • list(map(list,zip(*[[1, 2, 3], [4, 5, 6]]))) 是 [[1, 4], [2, 5], [3, 6]]

    • map(list,zip(*[[1, 2, 3], [4, 5, 6]])) 表示对下面zip对[[1, 2, 3], [4, 5, 6]] 变成键值对 然后list转换 最后返回一个迭代器对象 可输出[1, 4],[2, 5],[3, 6]

  • enumerate([3,5,7,3,7]) 表示下标和列表中对应的值 形成键值对 0:3 1:5 等等

  • __name__ 用来表示程序运行的方式 被其他模块导入时，就是本身模块名。自身运行时，就是__main__

  • 为什么应尽量从列表的尾部进行元素的增加与删除操作？

    • 当列表增加或删除元素时，列表对象自动进行内存扩展或收缩，从而保证元素之间没有缝隙，但这涉及到列表元素的移动，效率较低，应尽量从列表尾部进行元素的增加与删除操作以提高处理速度。

  • 简单解释Python的字符串驻留机制

    • Python支持字符串驻留机制，即：对于短字符串，将其赋值给多个不同的对象时，内存中只有一个副本，多个对象共享该副本。这一点不适用于长字符串
    • id(常字符串)==id(长字符串) 字符串相同但是依旧返回False

  • 集合a<集合b 判断a是否为b的真子集

  • 42、 使用切片操作在列表对象x的开始处增加一个元素3的代码为 x[0:0] = [3]

  • 已知 path = r’c:\test.html’，那么表达式 path[:-4]+’htm’ 的值为 ‘c:\test.htm’

  • abs(3+4j)值为5 3^2+4^2然后开根号

  • 对于带有else子句的for循环和while循环，当循环因循环条件不成立而自然结束时会执行else中的代码

  • 表达式 ‘%c’%65 的值为 ‘A’

  • 表达式 ‘:’.join(‘abcdefg’.split(‘cd’)) 的值为 ‘ab:efg’

  • Python swapcase() 方法用于对字符串的大小写字母进行转换。小写转大写 大写转小写

  • .endswith 参数可以是列表

  • Python ljust() 方法返回一个原字符串左对齐,并使用空格填充至指定长度的新字符串。如果指定的长度小于原字符串的长度则返回原字符串

  • round(int) 就是四舍五入

  • 表达式 ‘a’.join(‘abc’.partition(‘a’)) 的值为 ‘aaabc’

    • partition就是分割 但不删除a 分为 (‘’, ‘a’, ‘bc’) 然后 “”+a+a+a+”bc”

  • re.split(‘\.+’, ‘alpha.beta…gamma..delta’) 以点分割

  • 表达式 {1, 2, 3, 4, 5} ^ {4, 5, 6, 7} 的值为{1, 2, 3, 6, 7} 就是并集

  • 已知 x = [[1]] * 3，那么执行语句 x[0][0] = 5之后，变量x的值为[[5], [5], [5]]

    • *是应用 他们指向了同一个内存

  • 已知 table = ‘’.maketrans(‘abcw’, ‘xyzc’)，那么表达式 ‘Hellow world’.translate(table) 的值为 ‘Helloc corld’

    • maketrans() 方法用于创建字符映射的转换表，对于接受两个参数的最简单的调用方式，第一个参数是字符串，表示需要转换的字符，第二个参数也是字符串表示转换的目标

  • 已知字符串 x = ‘hello world’，那么执行语句 x.replace(‘hello’, ‘hi’) 之后，x的值为 ‘hello world’

    • replace 是返回修改后的值 不是修改x

  • Python标准库random中的sample(seq, k)方法作用是从序列中选择不重复的K个元素

  • random模块中shuffle()方法的作用是将列表中的元素随机乱序。

  • 对文件进行写入操作之后，flush()方法用来在不关闭文件对象的情况下将缓冲区内容写入文件。

  • 表达式 {1, 2, 3} == {1, 3, 2} 的值为 True

    • 字典无序 创建字典会自己排序

  • x = x.reverse() 之后，x的值为None 因为reverse是原地翻转

  • 表达式list(str([3, 4])) == [3, 4]的值为 False

    • list(str([3,4])是[‘[‘, ‘3’, ‘,’, ‘ ‘, ‘4’, ‘]’]

  • 表达式sorted({‘a’:3, ‘b’:9, ‘c’:78})的值为 [‘a’, ‘b’, ‘c’]

  • 表达式sorted({‘a’:3, ‘b’:9, ‘c’:78}.values())的值为 [3, 9, 78]

  • Python字典和集合属于无序序列

  • 对于生成器对象x = (3 for i in range(5))，连续两次执行list(x)的结果是一样的。（错）

    • 比如x=” “就是False ‘’.join(x.split())的值是 “”

  • 表达式 {1, 2} * 2 的值为 {1, 2, 1, 2}。错 会直接报错

  • 假设random模块已导入，那么表达式 random.sample(range(10), 20) 的作用是生成20个不重复的整数

    • 会报错 因为选项不足20

  • list没有find

  • print（0.1+0.2==0.3）是False 因为0.1+0.2 是约等于0.3 是有误差的

  • a=1;b=2 这样写可以

  • Python语言的浮点数必须带小数部分

  • Python capitalize()将字符串的第一个字母变成大写,其他字母变小写。对于 8 位字节编码需要根据本地环境。

    • Python函数定义中没有对参数指定类型，函数中应根据实参的类型使用

  • 6.4 简单解释Python中以下划线开头的变量名特点。

  • 在Python中，以下划线开头的变量名有特殊的含义，尤其是在类的定义中。用下划线作为变量前缀和后缀来表示类的- 特殊成员：

    • _xxx：这样的对象叫做保护变量，不能用’from module import*’导入，只有类对象和子类对象能访问这些变量；
    • __xxx__：系统定义的特殊成员名字；
    • __xxx：类中的私有成员，只有类对象自己能访问，子类对象也不能访问到这个成员，但在对象外部可以通过“-对象名._类名__xxx”这样的特殊方式来访问。Python中没有纯粹的C++意义上的私有成员。
