python特性

  • 强类型：不允许不同类型相加。例如：整形+字符串会报类型错误。

  • 动态：不使用显示数据类型声明，且确定一个变量的类型是在第一次给它赋值的时候。

  • 脚本语言：一般是解释性语言，运行代码只需要一个解释器，不需要编译。

    编译器

  • 计算机只能识别机器语言，必须把其他语言翻译成机器语言的工具，被称为==编译器==

  • 编译器翻译有两种 一种是编译，一种是解释，两种方式之间的区别在于翻译时间点的不同，前者执行前全部编译生成可执行文件，后者运行时解释一句源代码执行一句。

  • rm -r ~/.Pycharmxxx.x 删除初始配置 也就可以恢复pycharm的初始设置

  • /opt 给主机额外安装软件摆放的目录 （一个用户安装，多个用户使用）

  • linux下删除pycharm

    • 删除加压缩的目录
    • 删除家用户目录下保存配置信息的隐藏目录
    • /usr/share/applications/下的jetbrains-pycharm.desktop删掉

      基础知识

  • python的原程序 就是一个特殊格式的文本文件

  • 注释

    • # 单行注释 "“” "“” 多行注释

  • 运算符

    • 算术运算符
      • 最普通的加减乘除
      • python中运算符 **幂 //整除 %取余数
      • 在算术运算符**最高级 +-最低 其他都是中等
    • 比较(关系)运算符
      • == != >< >= <=
    • 逻辑运算符
      • and or not
    • 赋值运算符
      • += *=….. 等等
运算符 优先级
** 幂
* / % // 乘、除、取余数、取整除
+ - 加法、减法
<= < > >= 比较运算符

程序执行原理

  • 程序运行前先保存在硬盘上 当运行时加载的内存让CPU执行内存中的程序代码
  • python是先让python解释器的程序复制到内存中 让python解释器从上到下的翻译python程序中的代码 CPU负责执行翻译完成的代码

    变量

  • 在 Python 中，每个变量 在使用前都必须赋值，变量 赋值以后 该变量 才会被创建
  • python 数字只有int和float
    • a=75.0 不会像C语言精度降低
  • 布尔型参与运算 假是0 真是1
  • 格式化输出
    • %s字符串
    • %d 有符号十进制整数 %06d 表示输出的整数显示位数 不足的地方用0补全 超过就不变
    • %16d 16位不足补空格
    • %f 浮点数 %.02f表示小数点后显示两位
    • %% 输出%
  • 原始字符串最后不能加反斜杠”\”
  • 变量的命名由字母下划线数字组成 不能以数字开头 不能和关键词重名 区分大小写
  • 关键字就是已经被python内部使用的标识符 有特殊的含义

    if判断

  • 在 Python 中，if 语句 就是用来进行判断的，格式如下：
  • and 且 or 或 not 否

    循环

  • 程序发开有三个流程
    • 顺序 分支 循环
  • break 某一条件满足时，退出循环，不再执行后续重复的代码
  • continue 某一条件满足时，不执行后续重复的代码

    函数

  • 所谓函数，就是把 具有独立功能的代码块 组织为一个小模块，在需要的时候 调用
  • 函数的作用，在开发程序时，使用函数可以提高编写的效率以及代码的 重用

    函数的参数

  • 形参：定义 函数时，小括号中的参数，是用来接收参数用的，在函数内部 作为变量使用
  • 实参：调用 函数时，小括号中的参数，是用来把数据传递到 函数内部 用的
  • 无论传递的参数是 可变 还是 不可变，只要 针对参数 使用 赋值语句，会在 函数内部 修改 局部变量的引用，不会影响到 外部变量的引用
  • 缺省参数
    • 定义函数时，可以给 某个参数 指定一个默认值，具有默认值的参数就叫做 缺省参数
    • 调用函数时，如果没有传入 缺省参数 的值，则在函数内部使用定义函数时指定的 参数默认值
    • 在 调用函数时，如果有 多个缺省参数，需要指定参数名，这样解释器才能够知道参数的对应关系！
  • 多值参数
    • 参数名前增加 一个 * 可以接收 元组，参数名前增加 两个 * 可以接收 字典
  • 拆包
  • 没默认值的必须放在左边
  • 默认值参数的赋值只会在函数定义时被解释一次。当使用可变序列作为参数默认值时，一定要谨慎操作.只要默认值是可变对象，并且未传入这个参数，这个类的这个参数，就会指向函数给默认值开辟的空间

    使用模块中的函数

  • 模块 就好比是 工具包，要想使用这个工具包中的工具，就需要 导入 import 这个模块。命名规则遵循变量规则
  • .pyc 是编译过得py文件 可以提高执行效率 。程序目录下一班有个__pycache__目录 pyc文件是由python解释器讲模块的源代码转化为字节码

    字节码

  • python解释器解释源程序分成两个步骤 先处理源代码生成二进制字节码 然后对字节码进行处理才生成CPU可以识别的机器码
  • 有了模块的字节码文件之后，下一次运行程序时，如果在 上次保存字节码之后 没有修改过源代码，Python 将会加载 .pyc 文件并跳过编译这个步骤
  • 当 Python 重编译时，它会自动检查源文件和字节码文件的时间戳，如果你又修改了源代码，下次程序运行时，字节码将自动重新创建

高级变量类型

list

  • list.index(xx)查找值为xx的index
  • list.insert(a,b)在index为a插入b
  • list.extend(a)把==可迭代的a== 放到list里面 （列表 元组 字典都可以）
  • list.clear 一次性全部删除
  • list.remove 删除指定 (r如果有多个,删除遇到的第一个)
  • list.pop 弹出 也可以参数写index
  • list.count 统计列表某一个元素出现的次数
  • list.sort() 升序排序 需要降序 就参数写 reverse=True
  • list.reverse() 逆序 反序

元组

  • 不可修改 保护数据安全
  • 如果元组只有一个元素记得a=(5,) 写个逗号
  • tuple.count() 获取某个元素出现的次数
  • tuple.index() 找某个元素的下标
  • 格式化字符串后的() 本质就是一个元组
  • “%s”% tuple 结果是返回一个字符串

字典

  • 列表有序 字典无序
  • key是索引 value是数据
  • 键必须唯一
  • 值可以是任何数据类型 但是键只能使用字符串，数字或元组
  • dict.pop() 删除指定key
  • dict.update() 合并字典 合并的键值对包涵已经有的 那就覆盖
  • dict.clear() 清空

字符串

  • string.isspace() 只是否全是空格 \t\n\r 都可以返回True

  • string.isalpha() 是否全是字母

  • string.istitle() 是否是标题 （所有单词搜首字母大写）

  • string.islower() 至少有一个字母 且全部都是小写就返回True

  • string.startwith(str) 是否str开头

  • string 判断数字

    • 都无法判断小数
    • isdecimal() 只能判断数字
    • isdigit() 判断数据 (1) \u00b2 也就是unicode字符串 也可以
    • isnumeric() 可以判断unicode字符串 还可以 判断中文数字 比如 一千零一

  • string.find(str,start=0,end=len(string)) 查找字符串返回下标 找不到就返回-1

  • string.replace(old,new,count)

  • string.center() 默认使用英文的空格

  • string.split(str=””,num) str默认是\r\t\b和空格

公共方法

  • 字典不能比较大小
  • 字典不能切片 连顺序都没有咋切片
  • [1,2] *5 也是可以重复的 元组也可以这样用
  • 列表相加可以扩展
  • # TODO 接下来要做的事
  • 修改变量名技巧: 右键选择变量名 Refactor -> rename -> 第一个选项

变量细节问题

  • 变量和数据分开存储

  • 数据保存在内存的一个位置

  • 变量保存数据在内存的地址

  • 变量纪录数据的地址就叫做引用

  • 当修改不可变得变量时 不是修改数据 而是修改了数据的应用的指向

  • 只有列表和字典是可变类型 是可以修改内存里的数据 通过方法调用

  • 字典的key必须能是不可变型的

  • 局部变量在函数执行才会被创建 函数执行结束后局部变量会被系统挥手

  • 针对函数参数使用==赋值语句== 不会影响外部变量

  • 如果是参数内部的函数可以改变可变参数的值

  • 对于list +=是extend 其他都是先加后复制

    局部变量和全局变量

  • 局部变量的生命周期：局部变量 在 函数执行时 才会被创建，函数执行结束后 局部变量 被系统回收

  • 在处理全局变量的时候

    • 全局变量 是在 函数外部定义 的变量（没有定义在某一个函数内），所有函数 内部 都可以使用这个变量
    • 在函数内部，可以 通过全局变量的引用获取对应的数据，但是，不允许直接修改全局变量的引用 ，使用赋值语句修改全局变量的值
    • 如果在函数中需要修改全局变量，需要使用 global 进行声明

面对对象

  • 面向对象
    • 面向对象是更大的封装，根据职责在一个对象 封装多个方法
      • 类 是对一群具有 相同 特征 或者 行为 的事物的一个统称，是抽象的，不能直接使用。类 就相当于制造飞机时的图纸，是一个 模板，是 负责创建对象的
      • 对象是 由类创建出来的一个具体存在，可以直接使用
      • 对象名变量中 仍然记录的是 对象在内存中的地址
  • dir(obj) 可以查看obj所有的内置方法
  • python创建对象后 对象名还是记录对象在内存的地址
  • 对象.属性名=xxx 就可以给类增加属性
  • __xxx__ 对象的内置方法
  • self.xxx在对象中调用对象的内置方法或属性
  • 创建对象 先__new__ 然后 __init__
  • 销毁对象调用__del__ 程序结束清对象内存被也会调用
  • __str__可以控制对象输出信息（必须返回一个字符串）
  • x is y 类似id(x)==id(y) ==用来判断值 is判断是否指向同一个对象
  • 私有属性
    • __属性/方法 那就是私有属性或私有方法 (只是改了名字)

继承

  • ojbect是所有类的基类

  • 子类有父类所有属性和方法

  • 子类调用父类除了super().xx 还可以父类名.xxx(self) 调用

  • 子类对象方法 不能访问父类的私有属性或者方法

  • Python 中子类对象可以通过父类的公有方法间接访问父类私有方法和私有属性

  • 多继承

    • 父类之间应该避免有重名的方法或属性
    • 如果还是重名了，可以在obj.__mro__ 里面存放了方法搜寻 可以在这里面看到属性的调用路径，先在哪个类中搜索，找到就执行，没找到就报错

  • python3都是新式类 都以object为基类

  • 多态

    • 前提1 继承 前提2 子类重写父类的方法
    • 不同的子类对象调用父类的方法产生不同的执行结果

  • 类的方法在内存中只有一份 属性是每个对象自己的内存空间

  • 类是一个特殊的对象 -> 类对象

    • 类在运行时会被加载到内存
    • 类对象可以由自己的属性和方法
    • 向上查找机制：obj.xxx 先在obj查找xxx 没找到就去obj的父类查找xxx
    • 在类方法调用类属性 类名.类属性（self.类属性不一定能调用到父类的属性，如果本身有就调用本身的属性）
    • 无法在子类使用super().xxx 修改父类的类属性 只能查看
    • 类方法 可以用cls.访问类的属性 也可以通过cls. 调用其他的类方法 cls 指向调用这个方法的类
    • 类外部调用类方法 用类名.类方法 如果对象.对象的父类类方法 也可以

  • 类方法

    • 静态方法 既不访问类属性 也不访问实例属性
    • 可以类名.静态方法名

  • __new__ 先给对象分配空间然后返回对象的应用，解释器获得引用后作为第一个参数传给__ini__ 方法

    • __new__是一个类方法，我们在定义一个类方法时需要在函数前打上@classmethod装饰器，而__new__不需要，因为它经过特殊处理。
    • 重写__new__ 一定要 return super().__new__(cls)Python 的解释器 得不到 分配了空间的 对象引用，就不会调用对象的初始化方法

      异常

  • 程序在运行时，如果 Python 解释器 遇到 到一个错误，会停止程序的执行，并且提示一些错误信息，这就是 异常

  • 异常的传递: 当函数执行出现异常，会将异常传递给方法调用的一方 如果传递到主程序还没异常的处理就会程序终止

  • 抛出异常

    • 用 raise抛出 可以直接 raise str 抛出一个字符串

模块

  • 每一个以扩展名 py 结尾的 Python 源代码文件都是一个 模块
  • 导入某个模块的部分工具 from… import ….(全区变量 函数 类)
  • from .. import * 一次性导入所有工具（不推荐使用 重名不好排查）
  • 导入模块时 先在当前目录下搜索目录 然后才去系统目录搜索
  • 导入时会执行没有缩进的代码
  • __name__ 属性 其他模块导入时里面就是模块名 而在自己模块中就是__main__
  • 制作模块压缩包
    • 在目录下 创建setup.py
    • 在终端执行 python3 setup.py build 生成压缩包
    • python3 setup.py sdist 发布压缩包 然后就会生成一个压缩包
  • 模块卸载
    • 先找到对应的路径 module.__file__ 输出对应目录
    • 然后删除
  • 包
    • 一次导入多个文件
    • Package 是一个包含多个模块的特殊目录
    • 目录下有个特殊的文件__init__.py 要先在__init__.py 下导入目录下的模块
    • 包的命令方式和变量名一样

文件

访问方式 说明
r 以只读方式打开文件。文件的指针将会放在文件的开头，这是默认模式。如果文件不存在，抛出异常
w 以只写方式打开文件。如果文件存在会被覆盖。如果文件不存在，创建新文件
a 以追加方式打开文件。如果该文件已存在，文件指针将会放在文件的结尾。如果文件不存在，创建新文件进行写入
r+ 以读写方式打开文件。文件的指针将会放在文件的开头。如果文件不存在，抛出异常
w+ 以读写方式打开文件。如果文件存在会被覆盖。如果文件不存在，创建新文件
a+ 以读写方式打开文件。如果该文件已存在，文件指针将会放在文件的结尾。如果文件不存在，创建新文件进行写入
  • 读取大文件readline 一次读取一行
  • os.mkdir() 创建目录
  • os.path.isdir() 判断是否是目录
  • os.listdir() 返回所有文件夹

eval函数

  • 可以把字符串当有效的表达式来求值
  • 比如 eval(“1+2”) 返回3
  • 也可以写入cmd命令 比如写入 __import__(os).system(“cmd”)

    补充

  • random.randint(a.b) 返回int整数 包涵a，b
  • \t 字符表 协助输出文本是 垂直方向保持对齐
  • pycharm ctrl+q 可以弹出函数说明
  • 如果模块名是数字开头 pycharm就无法导入该模块
