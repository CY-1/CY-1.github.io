unbuntu基础知识点

  • 裸机 没有安装操作系统的计算机（裸机只能写机器语言）

  • 虚拟机 通过软件模拟完整的硬件系统功能的，运行在一个完全隔离环境中的完整的计算机系统

  • 单用户操作系统和多用户操作系统 区别于同一时间能有几个用户

  • linux目录结构

  • ubuntu只有一个根目录 所有文件都在里面

  • /home目录存下有所有用户目录

  • 隐藏文件第一个字符是 .

    基本命令

  • command [ -option ] [ parameter ] # []表示可有可无

  • 在man查看手册时 空格显示下一屏幕 回车滚动一行 b回滚一屏幕 f前滚一屏幕 q退出 /word 查询word

  • 显示所有文件
    • ls -a 显示所有文件 -l 用列表方式显示文件的详细信息
    • ls -l 时第一列有无d 表示是否是目录
    • ls -lha
    • ls通配符 * 任意个数数个字符 ?一个字符 []匹配里面一个字符组 [a-f]表示a到f的字符

  • 切换目录

    • (cd ~) 切换到当前用户的主目录 (cd -) 在当前和上次使用的俩目录可以相互切换

  • 创建文件夹

    • touch创建的文件如果已经存在可以修改文件的末次修改日期
    • mkdir可以递归创建目录 mkdir -p a/b/c/d
    • 目录和文件不可重名，在同一目录下

  • 删除文件

    • rm -f 强制删除 不会提示删除不存在的文件
    • rm -r * 可以删除所有文件和文件、

  • 复制文件

    • cp 源文件 目标文件 （复制文件）
    • cp 可以直接写cp 源文件 . 直接复制到当前目录（相同名字）
    • cp -i 会提醒如果有覆盖文件
    • cp -r 可以复制目录
    • mv 源文件 目标文件 移动文件
    • tree -d 只显示目录
  • 移动文件
    • mv 文件名 目录名：将文件移动到目标目录
    • 参数-f: 在 mv 操作要覆盖某已有的目标文件时不给任何指示
  • 显示文件
    • more和cat都是显示文件内容 cat是完整显示 more是分屏显示
    • cat -b 会输出行号 cat -n 空行也会输出行号
    • grep -n 显示匹配和行号 -v 求反 -i 忽略大小写（grep as 123.txt 查找123.txt中as存在的地方） 可以使用正则表达式|
  • echo
    • echo > 输出，会覆盖原文件 >> 会追加到文件中
    • 一个命令的输出可以通过管道作为另一个命令的输入
      • 比如 ls -l | more 让他分屏显示

  • shutdown -r now 立马重启

  • shutdown -c 就是取消关机

    远程

  • scp 远程拷贝文件 （scp -P 端口 username@remote:目录/文件名） 可以在端口号后加-r 表示复制文件夹

    • scp -P 端口 源文件名 username@remote:目录/ 把本机复制给服务器

  • FTP文件传输 用的端口默认 21

  • ssh-copy-id 用户名@IP 把本机的密匙传输给对应的计算机

    • ssh 分为公钥和私钥 公钥加密的文件需要私钥解密 私钥加密的文件就需要公钥解密

      用户

  • 硬连接数：有多少中方式直接访问该目录

  • chmod +/- rwx 文件名 +表示增加权限 -表示删除权限（也可以修改目录的权限）

  • groupadd 组名 添加组（需要sudo）

  • groupdel 组名 删除组

  • cat /etc/group 查看组信息

  • chgrp -R 组名 文件/目录名 递归修改文件或者文件夹 组权限

  • useradd -m -g 组 新建用户名（-m 自动创建用户家目录 -g 指定用户存在的组 不然就创建一个同名的组）

  • passwd 用户名 设置用户密码

  • userdel -r 用户名 删除用户 以及对应的home下目录

  • id 用户名 查询uid gid 组代号等

  • usemod设置用户的主组、附加组和登录shell

    • 附加组也就是/etc/group最后分号后的用户，可以拥有当前组的权限
    • sudo usemod -G sudo zhangsan 然后zhangsan就可以有sudo权限了 添加到sudo的附加组了
    • ubuntu登录后默认用的是dash xshell远程登录普通用户用的bash 其他用户dash需要sudo usermod -s /bin/bash

  • /etc/passwd 是保存用户信息的文件

  • /usr/bin/passwd 是用于修改用户密码的程序

  • which 查看命令所在的位置

  • /bin 主要是具体应用 /sbin 系统管理员专用的二进制代码存放目录 主要用于系统管理

  • /usr/bin 放后期安装的软件 /usr/sbin 超级用户的一切管理程序

  • chown 修改文件拥有者

  • df -h 显示磁盘剩余空间

  • du -h 显示当前目录下的文件打下

  • ps aux 查看进程的详细状况 a选项是表示显示其他用户的进程 u表示显示进程的详细状态 x显示没有控制终端的程序

其他命令

  • find [路径] -name “*.py” 查找就安排指定目录下的文件 包括子目录

  • 软连接 类似于快捷方式

    • ln -s 被链接的源文件（写绝对路径，如果写相对路径在移动后出错） 链接文件

  • 硬连接

    • ln 被连接 连接文件

  • linux里 文件和文件数据分开存储的

    • 软链接里面存 源文件名 通过源文件名找到数据
    • 硬连接和源文件名一起指向文件数据

  • 打包压缩

    • tar 只负责打包 不负责压缩
    • tar -cvf 打包文件.tar 被打包文件 路径 （解压缩）
    • tar -xvf 打包文件.tar
    • gzip
    • tar -zcvf 打包文件.tar.gz 被打包文件 路径 就可以打包后自动压缩的到 .tar.gz
    • tar -zxvf 打包文件.tar 先解压缩 后解包
    • tar -jcvf 打包文件.tar.bz2 被打包文件 路径 就可以打包后自动压缩的到 .tar.bz2
    • jar -zxvf 打包文件.tar.bz2 先解压缩 后解包
    • 如果想解压到指定的目录（目录必须存在） 加文件名后-C选项 然后跟上对应的目录

VI文件编辑器

  • 命令模式下 U回退版本 ctrl+R 前进版本
  • 命令默认下 按下两次y 然后 数字+p 就可将光标行复制19行
  • 打开文件时 vim 文件名 +行数 打开后直接到对应行数
  • vi命令
    • h 向左 j 向下 k向上 l向右
    • w 向后移动一个单词 b 向前移动一个单词
    • $ 行首 ^ 行位
    • gg 文件顶部 G 文件末尾
    • 数字gg 数字G :s数字 跳转到对应的行数
    • Ctrl+b 向上翻页 Ctrl+f 向下翻页
    • H 屏幕顶部 M屏幕中间 L屏幕底部
    • vi中以空行区分段落 ｛ 上一个段落 ｝下一个段落
    • %在成对的括号间切换
    • 标记 mx x是任意一个字母 然后可以利用:x 然后回到标记位置(行删除 标记删除 如果同名会覆盖)
    • v 进入可视模式 可以进行文本选择 ctrl+v 会以垂直模式选择文本
    • u 撤销上次命令 CTRL+r 回复撤销的命令
    • x 删除选中的文本或者光标所在的文本 数字+x 删除指定数目的字符
    • d 删除（和移动光标所结合） dd删除当前行 D删除当前到行位
    • y 复制（和移动命令一起使用） yy 复制一行 p粘贴 （和系统剪切板的内容不能使用 ）
    • 数字+p 粘贴几次
    • d和x 就类似剪切操作 可以删除后复制
    • 替换 r替换光标的字符 R替换光标开始以后的任意字符
    • :%s/旧文本/新文本/g 全局替换
    • 如果先选中了了一部分文本 写下 ==s/旧文本/新文本/g== 就是局部替换
    • :%s/旧文本/新文本/gc 确认替换 就是替换前会确认一下
    • << 减少缩进 >> 增加缩进
    • . 重复命令
    • /str 从当前光标查找str的位置 n下一个 N上一个
    • 在命令默认下 o就是在当前行的前面查一个空行 O就是在行后插入一个空行
    • 先ctrl+v 选取代码块 然后I 写入# esc退出 就可全部加上注释
    • :sp 分屏 :vsp 纵向分屏 ctrl+w 切换窗口
