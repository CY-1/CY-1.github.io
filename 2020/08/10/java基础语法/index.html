一些入门知识

  • 跨平台原理
    • 平台：不同的操作系统
    • java可以在不同的平台运行， 通过JVM在不同的操作系统运行
  • JRE
    • java runtime enviroment
    • 是java运行时环境 包涵JVM和运行时需要的核心态库
  • JDK
    • JAVA Development kit
    • 包涵JRE和开发人员使用的工具
    • 其中开发工具: 编译工具:javac.ext和运行工具java.exe
    • 想要开发一个全新的程序 必须安装JDK

      常用dos命令

  • dir 输出当前目录下文件名
  • cd xxx 进去xxx目录
  • 盘符名: 进去某个盘符
  • clear 清屏

    java程序开发流程

  1. 编写程序 ->得到java源程序
  2. 编译程序 ->得到java字节码文件
  3. 运行程序
  • 编译文件：javac 文件名
  • 运行文件: hava 类名(cnd运行时 要到对应的目录)

    注释

  • 注释是在程序指定位置添加说明信息
  • 不参与运行
  • 单行注释 //
  • 多行注释 /* */
  • 文档注释 /** /

    关键字

    关键字描述

  • 被赋予特定含义的单词

    关键字的特点

  • 关键字全部字母都是小写
  • 代码编辑器有特殊的颜色标识

    常量

  • 在程序运行中其值不可以发生改变的量
  • 空常量不能直接输出的

    数据类型

    计算机存储单元

    数据类型

  • java是强类型语言，对于每一种数据都给出了明确的数据类型 不同的数据类型分配了不同的空间 所以他们表示的数据大小也是不一样的
  • 整数默认是int 浮点数默认是double类型

    变量

  • 在程序运行过程中可以改变的量
  • 本质上讲 变量时内存中一小块区域
  • 数据类型 变量名 = 变量值;
  • 不可重复定义一个变量名字
  • 变量使用前要初始化
  • long l= 1000000000;会报错 因为整数默认是int太大了
    • long l= 1000000000l; 加个l就可以了
    • 同理 float f=13.45;也要编程 float f=13.14F; 防止精度降低

      标识符

  • 就是给类 方法 变量等起名字的符号
  • 规则
    1. 数字 字母下划线 美元符组成
    2. 不能以数组开头
    3. 不能使用关键字
    4. 区分大小写
  • 命名约定

    类型转换

    自动类型转换

  • 把表示数据范围小的值或变量赋值给另外一个数据范围大的变量
    • double d=10;
    • 只能按照下图自动类型转换

      强制类型转换

  • 把表示数据范围大的值或变量赋值给另外一个数据范围小的变量
    • int k = (int)88.88

      运算符和表达式

      数字

  • 运算符: 对常量或者变量进行操作的符号
  • 表达式：用运算符把常量或者变量连接起来符号java语法的狮子就可以被称为表达式
  • 整数相除只能得到整数 必须浮点数参与才能返回小数
  • i=j=y;也可以

    字符的操作

  • char b= i+c; 不对 因为 i+c 是int类型
  • 字符串的+操作
    • 字符串进行+操作就是拼接
    • “a”+”b”=”ab” “a”+66=”a66”

      赋值运算符

  • “+=” 注意点
    • short s=10; s+=10;没问题 但是 s=s+10不行 因为short和int相加了
    • += 地城隐含了强制类型转换
    • -= /= %= *= 也有强制类型转换

循环语句

  • for 循环
    • 如果for循环中条件判断语句不写 默认是True
    • for循环中定义的变量只能在for循环中使用
  • while循环
  • do..while循环
  • 跳转语句
    • continue 跳过本次循环
    • break 终止循环

    • 快速生成main方法: psvm 回车
    • 快速生成输出语句：sout 回车
  • 内容复制键

      数组

  • 用户存储多个相同数据类型的储存模型
  • 下图效果一样 但是叫法不一样
  • java数组必须先初始化后才能使用
    • 初始化就是数组中的数组元素分配内存空间的首地址 并为每个数组元素赋值
  • 动态初始化
    • 默认添加初始值 整数是0 浮点数是0.0 布尔值是flase 字符是空字符 引用类型是null
  • 数组元素访问
    • 如果输出数组名arr就是内存空间的地址值
  • 内存分配
  • 多个数组指向相同
  • 数组静态初始化

    方法

  • 定义
  • 定义调用
  • 形参和实参
    • 形参：方法定义中的参数
    • 实参：调用方法传入的参数
  • 注意事项
    • 不能嵌套定义
    • void 可以写return但是不能在return后加任何数据
    • 方法调用的时候实参是int可以被自动转换为方法要求的double

      重载

  • 方法重载指同一个类中定义的多个方法之间的关系 满足下列条件的多个方法互相构成重载
    • 多个方法在一个类中
    • 多个方法有着一样的方法名
    • 多个方法的参数不同(类型不同 或者数量不同)
  • 特点
    • 重载仅对应方法的定义 与方法的调用无关
    • 重载和返回值无关 只能通过参数来判断是否重载
  • 参数传递
    • 基本类型：就实参和形参
    • 引用类型： 形参影响实参

      面向对象

      类和对象

  • 对象：万物皆对象
  • 类：是对现实生活中一类具有共同属性和行为的事务的抽象
    • 类是对象的数据类型
  • 属性：对象具有的各种特征 每个特征有自己的值
  • 行为： 对象能执行的操作
  • 对象和类的关系：类是对象的抽象 对象是类的实体

    类

  • 基本格式
  • new出来的对象是在堆内存 类变量有默认值
  • 对象内存图
  • 调用成员方法会放入到栈内存中

    成员变量和局部变量

  • 成员变量：类中方法外
  • 局部变量：方法中的变量
区别 成员变量 局部变量
类中位置不同 类中方法外 方法内或者方法申明上
内存中为不知 栈内存 堆内存
生命周期不同 随着对象存在而存在 随着对象消失而消失 随着方法的调用而存在 随着方法的调用结束而消失
初始化值不同 有默认初始化值 没有默认初始化值

    • 权限修饰符
    • 可以修饰成员变量或者方法
    • 被修饰的成员 只能被本类中访问
  • this关键字
    • 当局部变量和成员变量重名 来区别成员变量和局部变量
    • this.成员变量名
    • 如果没有重名 那么直接表示成员变量
    • 方法被那个类应用 就是表示那个对象
    • 内存原理

  • JDK中提供各种功能的类
  • java.lang下的类 不需要导包

    String

  • 不可修改 创建后被更改 可以被共享
  • String代表字符串 用双引号
  • 效果相当char[] 但底层是字节数组byte[]
    • JDK8前是字节数组
  • Java 中，char 类型数组可以直接使用数组名打印。char 类型的数组就相当于一个字符串。
    • 输出流 System.out 是 PrintStream 对象，PrintStream 有多个重载的 println 方法，其中一个就是 public void println(char[ ] x)，它会直接调用这个方法来打印字符数组。因此可以直接打印出数组内容，而不是地址。
  • 构造方法
方法名 说明
public String() 创建一个空白字符串
public String(char[] chs) 根据字符数组
public String(byte[] by) 根据字符数组
public s=”abc” 直接赋值
  • String对象特点
    • 妹子new创建的对象内存地址都不同
    • 以””创建的字符串 只要字符相同 无论在代码中出现几次都指向一个String对象 在字符串池维护
  • 字符串的比较
    • 基本类型：是比较数据值是否相同
    • 引用类型：比较的是地址
    • 字符串要通过equals()比较

StringBuilder

  • 可变的字符串类 可以看做一个容器
  • append(任意类型) 添加数据 返回对象本身
  • reverse 返回相反的字符序列
  • StringBuilder和String相互转换
    • StringBuilder.toString()内置方法转化为String
    • StringBulder(String) 使用构造方法
