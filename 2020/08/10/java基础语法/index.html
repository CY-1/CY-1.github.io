一些入门知识

  • 跨平台原理
    • 平台：不同的操作系统
    • java可以在不同的平台运行， 通过JVM在不同的操作系统运行
  • JRE
    • java runtime enviroment
    • 是java运行时环境 包涵JVM和运行时需要的核心态库
  • JDK
    • JAVA Development kit
    • 包涵JRE和开发人员使用的工具
    • 其中开发工具: 编译工具:javac.ext和运行工具java.exe
    • 想要开发一个全新的程序 必须安装JDK

      常用dos命令

  • dir 输出当前目录下文件名
  • cd xxx 进去xxx目录
  • 盘符名: 进去某个盘符
  • clear 清屏

    java程序开发流程

  1. 编写程序 ->得到java源程序
  2. 编译程序 ->得到java字节码文件
  3. 运行程序
  • 编译文件：javac 文件名
  • 运行文件: hava 类名(cnd运行时 要到对应的目录)

    注释

  • 注释是在程序指定位置添加说明信息
  • 不参与运行
  • 单行注释 //
  • 多行注释 /* */
  • 文档注释 /** /

    关键字

    关键字描述

  • 被赋予特定含义的单词

    关键字的特点

  • 关键字全部字母都是小写
  • 代码编辑器有特殊的颜色标识

    常量

  • 在程序运行中其值不可以发生改变的量
  • 空常量不能直接输出的

    数据类型

    计算机存储单元

    数据类型

  • java是强类型语言，对于每一种数据都给出了明确的数据类型 不同的数据类型分配了不同的空间 所以他们表示的数据大小也是不一样的
  • 整数默认是int 浮点数默认是double类型

    变量

  • 在程序运行过程中可以改变的量
  • 本质上讲 变量时内存中一小块区域
  • 数据类型 变量名 = 变量值;
  • 不可重复定义一个变量名字
  • 变量使用前要初始化
  • long l= 1000000000;会报错 因为整数默认是int太大了
    • long l= 1000000000l; 加个l就可以了
    • 同理 float f=13.45;也要编程 float f=13.14F; 防止精度降低

      标识符

  • 就是给类 方法 变量等起名字的符号
  • 规则
    1. 数字 字母下划线 美元符组成
    2. 不能以数组开头
    3. 不能使用关键字
    4. 区分大小写
  • 命名约定

    类型转换

    自动类型转换

  • 把表示数据范围小的值或变量赋值给另外一个数据范围大的变量
    • double d=10;
    • 只能按照下图自动类型转换

      强制类型转换

  • 把表示数据范围大的值或变量赋值给另外一个数据范围小的变量
    • int k = (int)88.88

      运算符和表达式

      数字

  • 运算符: 对常量或者变量进行操作的符号
  • 表达式：用运算符把常量或者变量连接起来符号java语法的狮子就可以被称为表达式
  • 整数相除只能得到整数 必须浮点数参与才能返回小数
  • i=j=y;也可以

    字符的操作

  • char b= i+c; 不对 因为 i+c 是int类型
  • 字符串的+操作
    • 字符串进行+操作就是拼接
    • “a”+”b”=”ab” “a”+66=”a66”

      赋值运算符

  • “+=” 注意点
    • short s=10; s+=10;没问题 但是 s=s+10不行 因为short和int相加了
    • += 地城隐含了强制类型转换
    • -= /= %= *= 也有强制类型转换
